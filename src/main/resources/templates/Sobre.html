Motivação

Projeto da disciplina de Inteligência Artifial para tentar identificar o grau de dificuldade que estudantes teriam ao aprender uma nova linguagem de programação.

Para isso foi feito um questionário para identificar as dificuldade dos desenvolvedores ao aprender outra linguagem e encaminhado para diversos programadores preencherem. Com base nesses dados coletados, treinar uma Rede Neural Artificial para identificar o grau de dificuldade que será encontrado.
